Upload Bulk Product Data

Example File Format

Ensure your file is in one of the following formats: JSON, CSV, or TXT.

JSON:
[
    {
        "product_id": "12345",
        "category_id": "1",
        "product_name": "Sample Product",
        "product_price": "100.00",
        "product_quantity": "10",
        "product_image": "images/sample.jpg",
        "product_description": "This is a sample product description."
    },
    ...
]

CSV:
product_id,category_id,product_name,product_price,product_quantity,product_image,product_description
12345,1,Sample Product,100.00,10,images/sample.jpg,This is a sample product description.
...

TXT:
Each line should contain a product in the following format:
product_id|category_id|product_name|product_price|product_quantity|product_image|product_description
12345|1|Sample Product|100.00|10|images/sample.jpg|This is a sample product description.
...
